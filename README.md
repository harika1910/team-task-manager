# Team Task Manager

This is a full-stack web application that helps users manage projects and tasks easily.  
Users can create projects, assign tasks, and track progress in a simple dashboard.

---

##  What this project does

- Users can **signup and login**
- Admin can **create projects**
- Users can **create tasks**
- Tasks can be **marked as completed**
- Dashboard shows **total, completed, and pending tasks**

---

##  Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask)  
- Database: SQLite  

---

##  Features

###  Authentication
- Signup with name, email, password, role
- Login with credentials

###  Project Management
- Create new projects

###  Task Management
- Create tasks with title and description
- Assign tasks to users
- Mark tasks as completed

###  Dashboard
- View:
  - Total tasks
  - Completed tasks
  - Pending tasks

---

##  How to run this project

1. Clone the repository:

   ```
   git clone https://github.com/harika1910/team-task-manager.git
   ```

3. Go into the folder:

   ```
   cd team-task-manager
   ```

4. Install required packages:

   ```
   pip install -r requirements.txt
   ```

5. Run the app:

   ```
   python app.py
   ```

6. Open in browser:

   ```
   http://127.0.0.1:5000/
   ```

---

##  Project Flow

1. Signup → create account
2. Login → enter dashboard
3. Create project
4. Create tasks
5. Mark tasks as completed
6. View dashboard stats
