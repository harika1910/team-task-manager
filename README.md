# Team Task Manager

## Features
- User Signup & Login
- Project Management
- Task Creation & Assignment
- Task Status Tracking
- Dashboard Analytics

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Database: SQLite

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   python app.py
